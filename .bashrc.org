* Aliases

** emacs

#+begin_src bash
  alias e="emacsclient -n -c"
#+end_src

* Paths

Add path for user-specific executable files

#+begin_src bash
  export PATH=~/.local/bin:$PATH
#+end_src

* EDITOR environment variable

#+begin_src bash
  EDITOR="emacsclient -c"
#+end_src

* Zoxide to navigate terminal

#+begin_src bash
  eval "$(zoxide init bash)"
#+end_src
